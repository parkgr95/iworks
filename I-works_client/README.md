## 시작하기

프로젝트를 로컬 환경에서 실행하는 방법에 대한 안내입니다.

1. **의존성 설치**

   ```bash
   npm install
   ``````

2. **개발 서버 실행**

   Port No. 5173

   ```bash
   npm run dev
   ```
